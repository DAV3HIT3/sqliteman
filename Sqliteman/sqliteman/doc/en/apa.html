How to obtain Sqliteman

MS Windows

Unfortunately I have no access to any MS Windows machine. You can find the sources for this version in the Sqliteman area on github. You should be able to build it.

Warning

Downloadable Sqliteman for MS Windows can be found on the various third party sites. You can not be sure what these packages contain. Everything is propably fine there but there is no content warranty of 3rd party packages and they may be older versions.

Linux

It's highly probable you have Sqliteman binary package prepared for your distribution. Use this way if it's possible.

Fresh packages for some distributions are prepared in dedicated repository thanks to the great Opensuse Build Service.

There is also a prebuilt Linux binary (generally a bit out of date with respect to the sources) in github here .

When you cannot find your binary package, try to compile Sqliteman from Source Code.

Mac OS X

Unfortunately I have no access to any Mac machine. Sqliteman compiles and runs fine on this platform but it has to be provided by third parties.

Other UNIXes

Some reports from users tell that it can be setup on Solaris, Opensolaris, *BSD. It's provided by 3rd parties again.

Source Code

For the sources of the latest version go to github, click on the Clone or Download button and select "Download ZIP".
